# FairLens — Dashboard

Interactive React + Vite audit dashboard for the FairLens platform.

## Dev Server

```bash
npm install
npm run dev
```

## Pages

| Route | Page |
|-------|------|
| `/` | Data Upload & Ingestion |
| `/overview` | Dataset Overview & Statistics |
| `/selection` | Feature Selection |
| `/eda` | Exploratory Data Analysis |
| `/imputation` | Imputation Strategy Comparison |
| `/features` | Modeling Prep |
| `/fairness` | Fairness Lab (Causal Inference + Mitigation) |
| `/model` | Model Training & Evaluation |
| `/results` | Results & Audit Export |

## Stack

React 18 · TypeScript · Vite · TailwindCSS · shadcn/ui · Recharts · Framer Motion
